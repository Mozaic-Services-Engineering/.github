# Mozaic Services Engineering

We build and operate the software, platform capabilities, and delivery tooling that support the Mozaic engineering estate.

Our work spans product applications, shared platform services, AI-enabled capability layers, infrastructure, and the GitHub-based operating model used to plan and deliver change across the estate.

## What We Work On

- product application engineering across multiple Mozaic app families
- shared platform and runtime services
- AI service, control-plane, and governed automation capabilities
- infrastructure, delivery workflows, and environment reliability
- local developer experience and engineering tooling

## How We Work

We use GitHub as the engineering source of truth for:

- issues and implementation tracking
- backlog and sprint planning
- parent and child work relationships
- pull request linkage
- durable engineering documentation
